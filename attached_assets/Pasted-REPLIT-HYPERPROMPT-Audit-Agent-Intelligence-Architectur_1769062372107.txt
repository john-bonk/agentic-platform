REPLIT HYPERPROMPT: Audit Agent Intelligence Architecture
Full-Stack AI Assistant System with Report Generation, Conversational Agent, and In-Context Report Editing

ARCHITECTURE OVERVIEW
The Audit Agent is a three-tier intelligence system:

Intelligence Model (getAIResponse function):
const getAIResponse = (userMessage: string): { content: string; resources?: ResourceReference[] } => {
  const lower = userMessage.toLowerCase();
  
  // Keyword-based response mapping:
  if (lower.includes("overdue")) → Return overdue tasks summary + Task resource
  if (lower.includes("summary") || lower.includes("sox testing")) → Return progress stats + Report resource
  if (lower.includes("evidence") || lower.includes("documentation")) → Return evidence status + Document resource
  if (lower.includes("high-risk") || lower.includes("priority")) → Return high-risk controls + Control resource
  if (lower.includes("qa") || lower.includes("review")) → Return QA summary + Report resource
  if (lower.includes("task") || lower.includes("assign")) → Confirm task creation + Task resource
  if (lower.includes("report") || lower.includes("executive")) → Return report summary + Report resource
  
  // Default fallback:
  return generic helpful response asking for clarification
};

PART 4: COMPONENT 3 - REPORT AGENT (In-Context Editor)
File: client/src/components/report-agent.tsx

This is the most sophisticated component - ~1000+ lines of intelligence logic.

Purpose:
Embedded in the Report view page
Allows users to modify generated reports via natural language
Searches controls, users, metrics in real-time
Adds/edits/removes sections and charts
Key Interfaces:
interface Message {
  id: string;
  role: "user" | "assistant";
  content: string;
  action?: {
    type: "add_section" | "edit_section" | "remove_section" | "search_result";
    data?: any;
  };
  timestamp: Date;
}
interface ChartSpec {
  type: "bar" | "pie" | "line";
  data: number[];
  labels?: string[];
  title?: string;
  position?: "above" | "below" | "inline";
}
interface ReportAgentProps {
  sections: ReportSection[];
  onAddSection: (section: Omit<ReportSection, "id">) => void;
  onEditSection: (sectionId: string, content: string) => void;
  onRemoveSection: (sectionId: string) => void;
  onAddChart: (sectionId: string, chartSpec: ChartSpec) => void;
  onRemoveChart: (sectionId: string, chartIndex: number) => void;
}
Intelligence Model (processQuery function):
The processQuery function (~700 lines) handles:

Search Operations:

"search users" / "search team" → Query users API, return matches
"search metrics" / "search stats" → Return dashboard metrics
"search controls" / "find controls" → Filter controls by name/ID/risk/status
Section Management:

"show sections" / "list sections" → List all sections with chart counts
"edit section X" → Set pending edit, wait for follow-up
"add section" / "create section" → Generate new section with title/content
"remove section X" / "delete section" → Remove section by number/name
Chart Generation:

Parse chart type: pie, bar, line from keywords
Parse time range: "last 7 days", "past 30 days" etc.
Generate contextual data based on metrics:
Control status breakdown → pie chart
Testing results → bar chart
Work completion over time → line chart
Risk distribution → pie chart
Findings by severity → bar chart
Section Finding Helper:

Matches by number: "section 1", "section #2", "sec 1"
Matches by name: "executive summary", "insights"
Matches by position: "first section", "last section"
Data Dependencies:
The ReportAgent fetches live data via React Query:

const { data: controls } = useQuery<Control[]>({ queryKey: ["/api/controls"] });
const { data: metrics } = useQuery<DashboardMetrics>({ queryKey: ["/api/dashboard/metrics"] });
const { data: users } = useQuery<User[]>({ queryKey: ["/api/users"] });

app.post("/api/generate-report", async (req, res) => {
  // 1. Validate request with Zod schema

    sections: llmOutput.sections,

    createdAt: llmOutput.createdAt,

export default function Home() {
  const [isAIPanelOpen, setIsAIPanelOpen] = useState(false);
  const [aiPanelPrompt, setAIPanelPrompt] = useState<string | undefined>();
  const handleOpenAIPanel = (prompt: string) => {
    setAIPanelPrompt(prompt);
    setIsAIPanelOpen(true);
  };
  return (
    <>
      {/* In your layout */}
      <AuditAgent onOpenPanel={handleOpenAIPanel} />
      
      {/* At the end, outside main content */}
      <AIAssistantPanel 
        isOpen={isAIPanelOpen}
        onClose={() => { setIsAIPanelOpen(false); setAIPanelPrompt(undefined); }}
        initialPrompt={aiPanelPrompt}
      />
    </>
  );
}
